# LabFlow 智序 - 实验室智能管理系统

## 设计目的和意义

针对高校实验室管理效率低、预约流程繁琐、资源分配不均等问题，设计并实现了一套完整的实验室智能管理系统。系统提供在线预约、审批管理、仪器管理等功能，大幅提升实验室管理效率，实现资源的优化配置与合理利用。

## 关键技术

后端采用 FastAPI + SQLAlchemy 2.0 异步架构，结合 MySQL 数据库保障数据可靠性；前端基于 Next.js 16 + React 19 构建响应式界面；AI 功能基于自然语言处理技术，集成智能预约推荐与统计报表 AI 总结两大模块。

## 作品特色

系统支持多角色协同操作，包含实验室负责人审批、仪器管理、通知推送等完整功能链；采用前后端分离架构与 JWT 认证机制确保安全；智能预约助手可理解自然语言需求并自动推荐最优实验室与时段，降低用户操作成本；同时提供 AI 统计总结功能，自动生成日报、周报、月报数据摘要，提升管理决策效率。
