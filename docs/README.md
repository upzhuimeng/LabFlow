## LabFlow | 智序

智能实验室管理系统

---

## 这是什么？

一个用于管理实验室预约和仪器的系统。

**能做什么？**
- 实验员：查看实验室、预约使用、查看仪器
- 管理员：管理实验室和仪器、管理用户权限
- 审批人：审批预约申请

---

## 核心概念

### 实验室 (Lab)
- 包含名称、地址、容纳人数等基本信息
- 拥有若干仪器
- 由实验室负责人管理

### 仪器 (Instrument)
- 归属于某个实验室
- 记录型号、厂商、采购日期、价格等信息

### 标签 (Tag)
- 用于对实验室进行分类
- 每个标签对应一个标签管理人
- 标签管理人有权限审批涉及该标签的所有实验室预约

### 用户 (User)
| 角色 | 能做什么 |
|------|----------|
| 超级管理员 | 最高权限，可设置管理员 |
| 管理员 | 管理实验室、仪器、负责人、标签 |
| 实验员 | 使用实验室和仪器 |

### 审批人
实验室负责人和标签管理人负责审批预约申请。

---

## 预约流程

当实验员提交预约申请后：

```
实验员提交预约
       ↓
一级审批：实验室负责人审核
       ↓ (通过)
二级审批：标签管理人审核
       ↓ (通过)
预约成功
```

**两种结局**
- 任一级别拒绝 → 预约被拒
- 两级都通过 → 预约通过

---

## 数据规范

- 删除操作均为**软删除**（数据仍保留，只标记为删除）
- 所有操作都有**日志记录**，可追溯
