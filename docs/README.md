## LabFlow | 智序

一个智能的实验室管理系统

### 功能介绍

- 实验室
    - 实验室预约使用和管理是主要核心功能
    - 每个实验室将存放一些仪器
    - 通过仪器和实验室功能将创建两个字段
        - 字段一：标签
            - 主要用于后续实验室二级负责人的指定和管理实验室
        - 字段二：关键词
            - 用于后续Agent读取并生成内容
- 仪器
    - 每个仪器都归属于一个实验室(备注：仪器不包括试管，烧杯等低价值仪器)
- 用户
    - 用户分为两类和一特殊
        - 特殊为超级管理员，拥有一切权限，包括设置管理员，一般用于最高层的人员管理
        - 两类
            - 普通用户
                - 普通用户为实验室主要对象，拥有实验室和仪器的查看权限和实验室的预约权限
                - 部分普通用户将被添加到实验室的负责人名单中，作为实验室负责人(人员与实验员为多对多关系)，负责预约流程的第一级审批，对实验室负责
                - 更少量的用户将被添加实验室的标签，作为标签管理人（类似于系主任的职位，管理一个类别），负责第二级审批
            - 管理员
                - 管理员拥有仪器和实验室录入、修改和删除权限
                - 管理员为实验室设置和修改负责人和标签
                - 管理员可以赋予和移出用户标签
                - 管理员的设置只能由唯一的超级管理员设置
- 审批流程
    - 审批目前设定为两级
        - 一级为实验室管理员
        - 二级为标签管理员
- 其他
    - 上述提及的所有删除都以软删除的形式保留原有数据
    - 上述提及的所有操作都将记录在日志表中，便于后续溯源
